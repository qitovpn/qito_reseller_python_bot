QITO Net Redirect Link

The qito_net_redirect_link configuration is used in the bot's success message when users purchase QITO plans. This link will be displayed as a clickable redirect link in the purchase confirmation message.

ByPass Redirect Link

The bypass_redirect_link configuration is used in the bot's success message when users purchase ByPass plans. This link will be displayed as a clickable redirect link in the purchase confirmation message.

Important: Make sure the URLs are valid and accessible. Users will click on these links to access their account setup pages. QITO and ByPass plans can have different redirect URLs.
