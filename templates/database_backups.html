Backup Information
What are backups?

Database backups are timestamped copies of your database that include:

  • All user accounts and balances
  • VPN plans and keys
  • Payment history
  • Transaction records
Backup Management
  • Download: Save backup to your computer
  • Restore: Replace current database with backup
  • Delete: Remove backup file (permanent)
Warning: Restoring a database will replace all current data. Make sure to create a backup first!
